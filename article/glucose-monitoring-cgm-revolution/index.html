血糖・脂質 ・2024-12-05 ・読了5分

持続血糖測定（CGM）が変える糖尿病ケア — 「見えない血糖」を可視化して合併症を防ぐ

FreeStyleリブレ・Dexcom G7等のCGM（持続血糖測定）が保険適用拡大で一般化。HbA1cだけでは見えない「血糖変動」「食後高血糖」「夜間低血糖」を可視化し、食事・運動・薬調整に直結させる新時代の糖尿病ケアを解説。

「HbA1c 7.0% だから大丈夫」──そう安心していませんか？ 同じHbA1cでも、血糖変動が大きい人は合併症リスクが2-3倍高いことが分かってきました。2024年、持続血糖測定（CGM：Continuous Glucose Monitoring）が保険適用拡大で一般化し、「見えない血糖」を可視化する時代が来ました。

CGM（持続血糖測定）とは

皮下組織液中のグルコース濃度を5-15分間隔で連続測定し、スマホ・リーダー・受信機にリアルタイム表示するデバイスです。

CGMが見せる「HbA1cでは見えない3つのリスク」

1. 血糖変動幅（GV：Glucose Variability）

同じHbA1c 7.0%でも、血糖が80-180 mg/dLで推移する人と、50-300 mg/dLで乱高下する人では、後者の心血管イベントリスクが2.5倍（DCCT/EDIC追跡解析）。

  • 目標：CV（変動係数）<36%、MAGE（平均振幅）<80 mg/dL
  • 変動大 → 酸化ストレス・内皮機能障害・動脈硬化促進

2. 食後高血糖（Postprandial Glucose Excursion）

HbA1c正常でも、食後1時間で180 mg/dL超えが頻回なら、動脈硬化・認知機能低下リスク上昇（DECODE研究等）。

  • 目標：食後1時間 <180 mg/dL、食後2時間 <140 mg/dL
  • 食事内容・順序・運動タイミングで即座に改善可能

3. 夜間低血糖（Nocturnal Hypoglycemia）

無自覚の夜間低血糖（<70 mg/dL）は、不整脈・認知機能低下・翌朝の高血糖（ソモジ効果）の原因。CGMなしでは検出困難。

  • 目標：夜間（0-6時）<70 mg/dL を 0分、<54 mg/dL を 0分
  • 就寝前スナック・基礎インスリン調整・CGMアラートで予防

CGM活用の「4つのアクション」

1. 食事の「血糖スコア」化

同じ食事でも人により血糖反応は異なる（PREDICT研究）。CGMで「自分の食後反応」をスコア化：

  • 白米おにぎり2個 → ピーク220 mg/dL → 「要改善」
  • 玄米おにぎり＋卵 → ピーク140 mg/dL → 「良好」
  • 同じ炭水化物量でも「食べる順序・組み合わせ・調理法」で50-80 mg/dL差

2. 運動の「血糖下げ効果」可視化

食後15分ウォーキングでピーク30-50 mg/dL抑制 → CGMでリアルタイム確認 → モチベーション持続

3. 薬剤調整のエビデンス化

  • 「朝食前高い」 → 夕食前速効型インスリン増量
  • 「夜間低血糖」 → 基礎インスリン減量・就寝前スナック
  • 「食後高血糖持続」 → 超速効型インスリン変更・DPP-4阻害薬追加

4. 行動変容のフィードバックループ

リアルタイム表示 → 即座の行動修正（水飲む・歩く・薬飲む） → 成功体験 → 習慣化

今日からできること

1. 主治医に「CGMの適応があるか」相談

  • インスリン使用中なら即適応の可能性大
  • 非使用でも「血糖変動が気になる」「夜間低血糖が心配」で相談可

2. まずは「食後1時間血糖」を自己測定（指先穿刺）で把握

  • 朝・昼・夕食後1時間、1週間記録
  • 180 mg/dL超えが多い食事を特定
  • 主治医に持参し、CGM適応・食事指導の材料に

3. 「食べる順序」を今日から変える

  • 野菜・たんぱく質（肉・魚・卵・大豆）→ 炭水化物（米・パン・麺）の順
  • これだけで食後ピーク20-40 mg/dL低下（複数RCTで確認）
  • CGMなしでも即実践・効果確認可能
